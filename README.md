## 📖 Overview

NovaStore is a full-stack e-commerce platform designed to provide a fast, secure, and user-friendly online shopping experience. It includes customer-facing features, an admin dashboard, inventory management, order tracking, and payment integration.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/novastore.git
```

### Navigate

```bash
cd novastore
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
